# Metro dialogs

This user control lib implement some dialogs at metro style:
 * About box dialog
 * Message dialogs
 * Open/save file dialog
 * Select folder dialog
 * Select items dialog
 * Splash screen dialog

Based on MahApps.Metro and Caliburn.Micro
